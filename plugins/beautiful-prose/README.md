# beautiful-prose

A hard-edged writing style contract for timeless, forceful English prose without modern AI tics.

## Skills

### beautiful-prose

A style contract (not a vibe) for producing prose that is clean, exact, muscular, readable at speed, and rewarding on reread.

**Triggers on:** Requests for prose or rewrites that must be clean, exact, concrete, and free of AI cadence, filler, or therapeutic tone

## Usage

Activate by prepending to any request:

```
Apply the Beautiful Prose skill.
Write a 250 word memo on higher standards.
```

Optional control tags:
- `REGISTER: founding_fathers | literary_modern | cold_steel | journalistic`
- `DENSITY: lean | standard | dense`
- `HEAT: cool | warm | hot`
- `LENGTH: micro | short | medium | long`

## Installation

Add to your Claude Code plugins or install from the marketplace.
