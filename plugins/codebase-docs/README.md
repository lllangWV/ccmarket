# codebase-docs

Standards and best practices for writing codebase documentation files.

## Skills

### agents-md-standards

Guidance for writing effective AGENTS.md (or CLAUDE.md) files following context engineering best practices.

**Triggers on:** Questions about AGENTS.md structure, CLAUDE.md best practices, onboarding AI agents to codebases

### spec-standards

Structure and patterns for writing specification documents in `specs/` directories.

**Triggers on:** Questions about writing specs, design documentation, architecture docs

## Installation

Add to your Claude Code plugins or install from the marketplace.
