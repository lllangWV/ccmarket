# brainstorming

Turn ideas into fully formed designs and specs through natural collaborative dialogue.

## Skills

### brainstorming

Helps refine ideas into complete designs by understanding project context, asking targeted questions one at a time, and presenting designs in small validated sections.

**Triggers on:** Creative work like creating features, building components, adding functionality, or modifying behavior

## Process

1. **Understanding the idea** - Check project state, ask questions one at a time
2. **Exploring approaches** - Propose 2-3 options with trade-offs and recommendations
3. **Presenting the design** - Break into 200-300 word sections, validate each

## Installation

Add to your Claude Code plugins or install from the marketplace.
